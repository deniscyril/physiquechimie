Phase 1
Cours — La relation fondamentale
Les trois grandeurs
$$ n = \frac{N}{N_A} \quad \Longleftrightarrow \quad N = n \times N_A \quad \Longleftrightarrow \quad N_A = \frac{N}{n} $$
nquantité de matière, en mole (mol) Nnombre d'entités (atomes, molécules, ions…) — sans unité NAconstante d'Avogadro = 6,0 × 10²³ mol⁻¹
Exemple 1 — Calculer n à partir de N
Dans un verre d'eau, il y a 8,0 × 10²³ molécules d'eau. Calcule la quantité de matière en molécules d'eau.
1
Expression littérale : $n = \dfrac{N}{N_A}$
2
Application numérique : $n = \dfrac{8{,}0 \times 10^{23}}{6{,}0 \times 10^{23}}$
3
Résultat : $\boxed{n = 1{,}3 \text{ mol}}$
Exemple 2 — Calculer N à partir de n
Un comprimé d'aspirine contient 2,5 × 10⁻³ mol de molécules d'aspirine. Calcule le nombre de molécules dans ce comprimé.
1
Expression littérale : $N = n \times N_A$
2
Application numérique : $N = 2{,}5 \times 10^{-3} \times 6{,}0 \times 10^{23}$
3
Résultat : $\boxed{N = 1{,}5 \times 10^{21} \text{ molécules}}$
